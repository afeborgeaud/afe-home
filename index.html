pydsm

pydsm is a Python tool for exact computation of seismic waveforms in spherically symmetric (1-D) transversely isotropic (TI) media using the direct solution method (DSM). pydsm is based on the Fortran codes of the DSM (kawai et al., 2006) compiled into a python library (using numpy.f2py).

This website's intention is to showcase results obtained using pydsm, and other tools based on pydsm, such as pytomo. For installation instructions and documentation for pydsm, see the pydsm Github page.
